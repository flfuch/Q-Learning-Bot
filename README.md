# Q-learning Bot

Implementation of a learning environment and a robot navigating in it, based on an epsilon-greedy one-step Q-learning algorithm as described in:
http://incompleteideas.net/book/first/ebook/node65.html

## Prerequisites


```
matplotlib
numpy
```
